# Gabuzomeu
Création et animation d'un objet de façon procédural 

## Generation parameters 
one extrude = 1 rig
### chest :
- Largeur : Scale Y
- longueur: Scace X
- epaisseur : Scale Z 
- Loop cut Axe Y
### wings :
- Nombre de sub division = nombre d'aille
- (sub division Z = sub division X)
- Tail : extrude X -
- Largeur : Scale Y
- longueur: Scale X
- epaisseur : Scale Z 
### head : extrude X +
- Largeur : Scale Y
- longueur: Scale X
- epaisseur : Scale Z 
- Nombre de subdivison pour les tantacules a l'arriere
